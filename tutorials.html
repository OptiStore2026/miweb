Solucion de problemas y guias

Netflix

Error de codigo 2FA por correo

Si ves el mensaje "Introduce el codigo que enviamos a tu correo" al iniciar sesion, usa la opcion Obtener ayuda y luego selecciona Iniciar con contrasena.

Referencia Netflix

Este flujo evita el bloqueo temporal del codigo y permite entrar con la credencial principal.

Si la cuenta sigue fallando, abre ticket en Discord en el canal de reemplazos.

CapCut

Error "Numero maximo de intentos alcanzado"

Este error no siempre significa que la cuenta este mala. Generalmente indica limite de intentos, bloqueo por IP o restriccion temporal del dispositivo.

Referencia CapCut
  • Puede no funcionar en la app movil.
  • Prueba iniciar desde navegador web: https://www.capcut.com/login
  • Si persiste, espera unos minutos y vuelve a intentar.

Nitro Promo Code

Requisitos importantes

Debes agregar un metodo de pago (tarjeta) para canjear el codigo. Discord lo pide para verificar la suscripcion.

Elegibilidad (solo usuarios nuevos)

El codigo Nitro solo funciona si no tuviste Nitro en los ultimos 12 meses.

  • Si usaste Nitro recientemente, el codigo no aplicara.
  • Verifica tu region y el tipo de promocion antes de reclamar.
